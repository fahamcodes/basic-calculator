# 🧮 Basic Calculator

A simple and responsive calculator built using **HTML, CSS, and JavaScript**.

This project performs basic arithmetic operations with a clean dark UI design.

---

## 🚀 Live Demo

🔗 https://fahamcodes.github.io/basic-calculator/

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- 🧼 Clear (CLR) button
- ⚡ Error handling
- 🎨 Clean Dark Theme UI
- 📱 Responsive Layout

## 🛠️ Technologies Used

- CSS3
